FAQ. Curious?

Q: Did you really make this website from scratch?
A:  Using Bootstrap and some CSS knowledge, I was able to create the backbone of this website. It wasn't easy though.
Q: What led you to create a website for the first time?
A:  It was generally out of pure interest. I had some prior knowledge with using HTML, and the idea of creating a beneficial piece of online architecture excited me. In addition, creating this project has allowed me to express myself freely in an eloquent manner.
Q: What do you want to accomplish in the next few years?
A:  I can't say I am confident in knowing what my goals are after university, but I would like to continue with disciplines similar to front-end development and possibly server-side. The fact that you can understand how websites are built to function responsively is simply phenomenal.
